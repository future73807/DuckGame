# 🦆 小黄鸭漂流记 - 项目说明文档

## 📖 项目简介

「小黄鸭漂流记」是一款基于 Web3D 技术开发的休闲收集类小游戏。玩家将扮演一只在水面冒险的小鸭子，通过灵活的走位收集道具获取积分，同时躲避障碍物。游戏支持键鼠与虚拟摇杆双端操作，并拥有昼夜交替、组合倍率、随机事件、难度递进与极光/流星等特色机制，适合碎片化时间娱乐。

> **在线体验**：[GitHub Pages](https://future73807.github.io/DuckGame/)（如有部署）

---

## 🚀 启动方式

| 命令               | 说明                                               |
| :----------------- | :------------------------------------------------- |
| `npm start`      | 启动后端服务器（推荐），访问 http://localhost:8123 |
| `npm run server` | 只启动后端（端口 8123）                            |
| `npm run client` | 只启动前端静态服务器（端口 5500，排行榜只读）      |

或直接双击 `start.bat` 一键启动。

> **注意**：`npm start` 和 `npm run server` 都只会启动后端服务器（端口 8123）。如需同时启动前端开发服务器，需另开终端运行 `npm run client`。

## 🎮 游戏特色

### 核心玩法

- **收集系统**：水草(+1分)、花朵(+2分)、荷叶(护盾+3)、磁铁(磁吸12秒)、血瓶(+1生命)
- **组合倍率**：集齐3个同色道具 → **10倍积分**10秒！集齐3个不同色 → **5倍积分**10秒！
- **生命系统**：初始3颗心，被漩涡/鲨鱼撞击扣心，生命归零游戏结束

### 环境系统

- **24小时昼夜循环**：凌晨极光、日出晨雾、中午鲸鱼云、日落晚霞、夜晚流星
- **随机事件**：顺风/逆风、暴风雨、彩虹祝福、水下暗影、海浪汹涌、道具雨、平静时刻
- **难度递进**：游戏时长越长，坏事件概率越高，漩涡生成越频繁

### 视听体验

- **自然音效引擎**：划水、收集、碰撞、护盾破碎等全部采用合成自然音效
- **镜头光晕**：动态追踪太阳位置的真实镜头光晕效果
- **漩涡吸入滤镜**：后处理全屏涡旋扭曲 + 旋转模糊 + 暗角收拢

### 社交功能

- **排行榜系统**：支持昵称、密码保护、全球排名
- **分享卡片**：生成精美塔罗牌式卡片，支持下载保存

## 🎯 游戏核心机制

### 操作方式

- **PC**：W/S 前进后退，A/D 相对视角左右平移，鼠标拖动旋转视角
- **移动端**：左半边屏幕滑动召唤摇杆，右半边屏幕滑动切换视角

### 道具系统

| 道具      | 效果                         |
| :-------- | :--------------------------- |
| 🌿 水草   | +1分                         |
| 🌸 花朵   | +2分                         |
| 🍃 荷叶   | 护盾+3（可抵挡一次伤害）     |
| 🪨 石头   | -1心（护盾可挡）             |
| 🧲 磁铁   | 磁吸12秒（自动吸引附近道具） |
| ❤️ 血瓶 | +1生命                       |

### 组合倍率机制（核心玩法）

- 集齐3个同色道具 → 触发 **10倍积分**，持续10秒！
- 集齐3个不同色道具 → 触发 **5倍积分**，持续10秒！

### 护盾机制

荷叶提供护盾，可抵消石头等负面效果。护盾破碎时播放真实玻璃碎裂音效。

### 生命系统

- 初始生命值为 3 颗心（❤️❤️❤️）
- 生命值归零时，游戏提前结束
- 被漩涡吸入、被鲨鱼撞击都会扣心
- 血瓶可恢复生命，上限 5 颗心

### 随机事件系统

每 30 秒触发一次全局事件，事件前 3 秒显示预警：

- 🌪️ **顺风/逆风**：移动速度变化
- 🌧️ **暴风雨**：视野缩小、巨浪、雷电、落石增多
- 🌈 **彩虹祝福**：花草额外 +5 分
- 🦈 **水下暗影**：鲨鱼追踪撞击
- 🌊 **海浪汹涌**：巨浪起伏
- 🎁 **道具雨**：血瓶荷叶从天而降
- ☀️ **平静时刻**：风平浪静

### 难度递进

- 游戏时长越长，坏事件概率越高
- 漩涡生成频率随时间增加
- 事件持续时间随难度延长
