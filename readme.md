# 🦆 小鸭子冒险 (3D Duck Adventure) - 项目说明文档

## 📖 项目简介
「小鸭子冒险」是一款基于 Web3D 技术开发的休闲收集类小游戏。玩家将扮演一只在水面冒险的小鸭子，通过灵活的走位收集道具获取积分，同时躲避障碍物。游戏支持键鼠与虚拟摇杆双端操作，并拥有昼夜交替、组合倍率与随机事件等特色机制，适合碎片化时间娱乐。

---

## 🏗️ 项目架构

### 1. 技术栈推测
基于文件特征（`3d-duck.html`及交互描述），项目采用纯前端单文件/轻量级架构：
*   **核心渲染引擎**：Three.js / Babylon.js 等 Web3D 引擎
*   **交互控制**：自定义虚拟摇杆 + PointerLock/Mouse 移动监听
*   **UI 层**：HTML5 + CSS3 (绝对定位/弹性布局覆盖于 Canvas 之上)
*   **状态管理**：原生 JavaScript 管理游戏状态（积分、护盾、时间、倍率、事件）

### 2. 核心模块划分
| 模块名称 | 功能职责 |
| :--- | :--- |
| **场景管理** | 3D场景初始化、水面渲染、天空盒/昼夜光照切换、物体生成与回收 |
| **角色控制** | 小鸭子模型控制、W/S/A/D键位移动（基于视角）、边界限制 |
| **相机控制** | 第三人称跟随、鼠标拖拽旋转视角 |
| **交互系统** | PC端键鼠事件监听、移动端触摸摇杆逻辑 |
| **道具与碰撞 (Prop & Physics)** | 道具生成逻辑、碰撞检测、积分/护盾/扣分结算 |
| **🎮 事件系统** | 随机全局事件的触发、倒计时管理、全局状态修正（如风力/暴雨等） |
| **游戏循环** | 60秒倒计时、状态更新、UI同步、游戏结束判定 |
| **UI管理 (HUD)** | 加载界面、积分/护盾/时间/事件状态显示、规则弹窗 |

### 3. 游戏核心机制（当前已实现）
*   **操作方式**：
    *   PC：W/S 前进后退，A/D 相对视角左右平移，鼠标拖动旋转视角。
    *   移动端：虚拟摇杆控制方向与速度。
*   **昼夜系统**：点击时钟图标切换昼夜，影响视觉表现。
*   **道具系统**：
    *   🌿 水草：+1分
    *   🌸 花朵：+2分
    *   🍃 荷叶：护盾+3
    *   🪨 石头：-1分
*   **组合倍率机制（核心玩法）**：
    *   集齐3个同色道具 → 触发 **10倍积分**，持续60秒！
    *   集齐3个不同色道具 → 触发 **5倍积分**，持续60秒！
*   **护盾机制**：荷叶提供护盾，可抵消石头等负面效果。

### ❤️ 2. 生命机制系统
**目标**：丰富游戏策略性，从“纯收集”过渡到“生存+收集”。
*   **基础设定**：
    *   初始生命值为 3 颗心（❤️❤️❤️）。
    *   生命值归零时，游戏提前结束（不受60秒限制）。
    *   当前期已有“护盾”时，受伤优先扣除护盾，护盾为0时扣心。
*   **扣心交互**：受击时屏幕红色闪边，小鸭子模型播放受击闪烁/硬直动画。

### 🌀 3. 扣心机制 - 海上漩涡
**目标**：引入动态环境障碍，提升游戏紧张感。
*   **视觉效果**：水面生成带有吸附动画的漩涡模型（粒子特效/Shader实现）。
*   **行为逻辑**：
    *   漩涡在场景中随机刷新，存在一定时间后消失。
    *   漩涡周围有**引力场**，小鸭子靠近时会被缓慢吸入（需玩家反向操作挣脱）。
    *   被吸入漩涡中心判定为死亡：扣除 1 颗心（❤️），并强制将小鸭子传送回安全出生点。

### ➕ 4. 加心道具
**目标**：提供容错率，增加随机惊喜。
*   **视觉设计**：漂浮在水面上的爱心/急救包/生命之果模型，散发微光。
*   **生成逻辑**：低概率随机生成（或在生命值≤1时提高生成概率，作为 mercy mechanism 仁慈机制）。
*   **拾取效果**：
    *   拾取后恢复 1 颗心（❤️），上限不超过初始最大生命值。
    *   播放治愈音效与+❤️的飘字特效。

### 🎲 5. 随机事件系统 (Event System)
**目标**：打破单局游戏的单调感，通过环境突变增加刺激感和策略深度。
*   **触发机制**：
    *   每局游戏（60秒）内随机触发 1-2 次全局事件。
    *   事件触发前 3 秒，屏幕上方显示预警图标和文字（如："⚠️ 暴风雨来袭"）。
*   **事件示例**：
    *   🌪️ **顺风/逆风**：场景出现风向特效，小鸭子向某方向移动速度增加，反方向移动变慢。
    *   🌧️ **暴风雨**：视野大幅缩小（相机拉近/加雾），水面波浪加剧，石头刷新率临时增加。
    *   🌈 **彩虹祝福**：全屏洒落彩虹特效，场上所有水草和花朵临时转化为高分道具，或基础积分 x2。
    *   🦈 **水下暗影**：水底出现追踪小鸭子的黑影，玩家必须保持移动，停留超过3秒会被撞击扣心。
*   **UI与反馈**：
    *   HUD 区域增加「当前事件状态栏」，显示事件图标与剩余时间倒计时。
    *   事件开始与结束时，播放专属的全屏过渡动画与音效。

### 🎵 6. 音效反馈
**目标**：提升游戏操作手感与沉浸感。
*   为收集水草、花朵、扣分、触发倍率、漩涡吸附、事件触发加入不同音效。

### 🦆 7. 小鸭子动画
**目标**：让角色更具生命力，动作反馈更直观。
*   增加游泳时的拨水动画、加速时的尾部水花特效、逆风时的吃力姿态。

### 📈 8. 难度递进
**目标**：保障单局游戏体验的曲线，越到后期越紧张。
*   随时间推移，漩涡生成频率增加、恶劣天气事件概率提升。
